##docker命令
```
docker ps ： 查看容器

docker images ：查看镜像

docker inspect ：查看网络

docker -run -it --name command 运行（交互式）

docker rmi/rm :删除

docker start/stop ：启动、停止

docker save/load import/export 导入、导出

docker login/logout 登录、退出

docker tag 标签

```
